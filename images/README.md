# Папка с фотографиями сайта

Положите сюда файлы **.jpg**, **.png** или **.webp** — сайт подхватит их автоматически после `npm run dev` или `npm run media`.

## Структура папок

| Папка | Что сюда класть | Имя файла |
|-------|-----------------|-----------|
| **hero/** | Фон главного экрана | `hero.jpg` (или любой один файл) |
| **people/** | Портреты руководства | `tarbokov.jpg`, `betuaganov.jpg` |
| **cofounders/** | Документы соучредителей (4 шт.) | `01.jpg`, `02.jpg`, `03.jpg`, `04.jpg` (по порядку) |
| **news/** | Фото к новостям | как `slug` новости, см. `src/data/news.ts` |
| **gallery/** | Все остальные фото | любые имена, сортировка по имени |
| **videos/** | Обложки «Золотой микрофон» | `01.jpg` … `04.jpg` (по порядку частей) |

## Новости — имя файла

В `src/data/news.ts` у каждой новости есть поле `slug`, например `ogni-dagestana`.

Фото кладите сюда: `news/ogni-dagestana.jpg` (то же имя, что `slug`).

## После добавления фото

```bash
npm run media
```

или просто перезапустите сайт:

```bash
npm run dev
```

## Тексты

- Новости (заголовки и текст): `src/data/news.ts`
- Подписи к документам соучредителям: `src/data/cofounders-docs.ts`
- Остальной текст: `src/data/content.ts`
